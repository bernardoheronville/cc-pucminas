## O que são Sistemas de Banco de Dados? Quais os componentes destes sistemas?
Sistema de Banco de Dados são um conjunto de dados relacionados e sua respectiva forma de acesso e organização. São compostos por uma coleção de dados organizados, uma estrutura lógica e um software que provê acesso aos dados a usuários e aplicações.
## Diferencie modelo, esquema e catálogo (dicionário) de dados.
Modelo é uma forma de representar o minimundo, esquema é a forma como os dados serão organizados, e o catálogo é o repositório de metadados que armazena a descrição do esquema.
## Defina independência de dados proporcionada por uma arquitetura em três esquemas
Independência de dados é a capacidade de se alterar o esquema em um nível de arquitetura sem precisar alterar os níveis superiores.
## O que é e para que serve SQL?
SQL é uma linguagem de consulta estruturada que combina VDL, DDL, SDL, e DML. Serve para definir as estruturas do banco de dados (tabelas, visões) e para dar instruções de manipulação, fazendo inserções, remoções, atualizações e consultas aos dados.
